# 本地训练指南

本项目可以在本地训练模型。当前有两条训练线：

1. CALiSol-23 电导率模型：用于 LiPF6、LiTFSI 等训练集中覆盖盐的电导率预测。
2. LiNO3 溶解度模型：用于硝酸锂在非水溶剂中的溶解度辅助评分。

## 一键重建

在项目根目录运行：

```powershell
.\scripts\setup_project.ps1
```

这个脚本会创建虚拟环境、安装依赖、同步公开数据并重新训练模型。

## 分步训练

如果环境已经装好，可以分步执行：

```powershell
$env:PYTHONPATH = (Get-Location).Path
.\.venv\Scripts\python.exe scripts\sync_public_data.py
.\.venv\Scripts\python.exe scripts\train_conductivity_model.py
.\.venv\Scripts\python.exe scripts\train_lino3_solubility_model.py
```

训练输出：

```text
models/conductivity_model.joblib
models/lino3_solubility_model.joblib
models/training_report.json
models/lino3_solubility_report.json
```

## 本地使用完整模型

本地启动脚本会显式启用完整电导率模型：

```powershell
.\scripts\start.ps1
```

等价环境变量：

```powershell
$env:IONMIX_CONDUCTIVITY_MODEL = "enabled"
```

## Render 免费档为什么默认不用完整电导率模型

Render 免费实例内存约 512 MB。完整 CALiSol-23 ExtraTrees 电导率模型在云端加载后可能超过这个限制，导致实例 502 或反复重启。

因此线上免费部署默认是轻量模式：

```text
IONMIX_CONDUCTIVITY_MODEL=disabled
```

轻量模式仍然会使用：

- 溶剂目录的分子描述符；
- 物理启发式溶剂化/传输评分；
- LiNO3 小溶解度模型；
- 闪点、黏度、安全性、稳定性约束；
- 二元/三元溶剂组合搜索。

如果升级到更高内存实例，可以在 Render 环境变量中设置：

```text
IONMIX_CONDUCTIVITY_MODEL=enabled
```

然后重新部署，恢复完整电导率模型。

## 继续提高模型质量需要什么数据

当前模型能做“实验前预筛选”，但不是最终配方证明。要提高可信度，最有价值的数据是：

- LiNO3 在二元/三元混合溶剂中的实测溶解度；
- 对应配方的电导率、黏度、温度依赖；
- 不同盐浓度下的电导率曲线；
- 电化学窗口、循环性能、界面兼容性；
- 失败样本：析晶、分层、黏度过高、不可燃性不达标等。

有这些数据后，可以把训练目标从“纯溶剂标签 + 启发式外推”升级到“混合配方监督学习”。
