# Electrolyte Explorer

一个可本地运行的电解液二元溶剂与配比预筛选系统。输入锂盐、温度、浓度、应用场景和目标权重，系统会搜索候选溶剂组合、优化起始比例，并输出评分、置信度、推荐理由和数据覆盖情况。

## 快速启动

本项目已经完成环境、公开数据同步和模型训练，直接在 PowerShell 中运行：

```powershell
.\scripts\start.ps1
```

浏览器打开 <http://127.0.0.1:8000>。

停止服务：

```powershell
.\scripts\stop.ps1
```

如果要从零重建环境、重新下载公开数据并训练模型：

```powershell
.\scripts\setup_project.ps1
```

运行自动测试：

```powershell
$env:PYTHONPATH = (Get-Location).Path
.\.venv\Scripts\python.exe -m pytest -q
```

## 系统能做什么

- 支持输入 `LiNO3`、`硝酸锂`、`LiPF6`、`LiTFSI` 等目标盐。
- 从 26 种电池相关溶剂中枚举二元组合。
- 对每个组合搜索 10:90 到 90:10、步长 5% 的比例。
- 五个目标权重总和固定为 100%，并按从上到下逐级锁定。调整某项时，上方项目不变，仅下方项目按比例分配剩余权重；若当前值将导致总和超过 100%，它会被限制在剩余额度内。
- 同时考虑盐溶剂化、离子传输、黏度、闪点、相态、电化学稳定性和低温表现。
- 对 CALiSol-23 覆盖的锂盐使用机器学习电导率预测。
- LiNO3 已加入独立的实测溶解度训练集：25 条记录、5 种溶剂，覆盖 25–45 °C。
- 每个结果展示为什么被推荐，而不是只返回一个黑箱总分。
- 候选依据用户目标权重计算综合评分，最终结果列表按综合评分从高到低展示；综合评分相同时按置信度排序。
- 当严格约束下候选不足时，仍返回低置信度备选，并逐项标明违反的约束。

## 当前模型结果

- 数据集：CALiSol-23
- 训练集：9,915 条
- 测试集：3,320 条
- 切分方式：按 DOI 分组，避免同一文献数据同时落入训练集和测试集
- 测试 MAE：1.1942
- 测试 R²：0.7406

目标 `k` 保留 CALiSol-23 原始定义和量纲。跨来源使用前应再次核对实验条件与单位。

LiNO3 溶解度子模型：

- 训练记录：25 条
- 独立溶剂：5 种
- 主要温度范围：25–45 °C
- 主要实验记录不确定度：约 5%
- 留一溶剂验证 MAE：约 0.80 log10 摩尔分数

留一溶剂验证误差较大，说明现有数据足以消除“完全无训练标签”的问题，但仍不足以支持高置信度地外推到任意新溶剂及二元混合物。因此系统把训练域距离和局部数据密度明确计入置信度。

## 重要边界

该系统是实验前预筛选工具，不是热力学溶解度计算器，也不直接证明某个配方可用于电池。对于 LiNO3，系统已有纯溶剂实测溶解度监督标签，但仍缺少足量二元混合物和电导率标签。因此输出的是“建议先做哪些实验、从什么比例开始”，不是已验证的最佳配方。最终必须测量：

- 实际溶解度与长期析晶；
- 离子电导率、黏度和温度依赖；
- 氧化/还原稳定窗口；
- 与正极、负极、隔膜和集流体的兼容性；
- 闪点、热稳定性和毒理风险；
- 扣式或软包电池循环结果。

## 数据来源

- [CALiSol-23](https://github.com/Pele0599/CALiSol-23)：锂盐电解液电导率数据。
- [Xin et al., Fluid Phase Equilibria 2018](https://doi.org/10.1016/j.fluid.2017.12.034)：LiNO3 等锂盐在非水溶剂中的实测溶解度。
- [PubChem PUG REST](https://pubchem.ncbi.nlm.nih.gov/docs/pug-rest)：化合物身份、CID、标准结构和基础分子属性。
- [RDKit](https://www.rdkit.org/docs/GettingStartedInPython.html)：SMILES 标准化与分子描述符计算。
